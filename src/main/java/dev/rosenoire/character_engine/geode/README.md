# What is Geode?

Geode is a library comporting multiple modules to help with java development and more specifically minecraft mod
development.

# Modules

| Module Name | Function                                                                                                                                                                                                                                                           |
|-------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Amethyst    | Math oriented features including handy datatypes like `double2`, `double3` as well as utility methods.                                                                                                                                                             |
| Ruby        | Module oriented around gizmos with the goal of making your debugging journey as smooth as possible.                                                                                                                                                                |
| Obsidian    | Minecraft centred module comporting utility methods relating to minecraft itself, as well as indexes such as: `BlockEntityIndex`, `BlockIndex`, `BlockTagIndex`, `EntityTypeIndex`, `GroupIndex`, `HudElementIndex`, `ItemIndex`, `ItemTagIndex` and `SoundIndex`. |
| Topaz       | Minecraft Data-Generation centred module, with the goal of making generating your resource files as trivial as possible.                                                                                                                                           |
| Lazuli      | Module centred around the minecraft library Cardinal Components API to expand it and bring high level utilities.                                                                                                                                                   |
